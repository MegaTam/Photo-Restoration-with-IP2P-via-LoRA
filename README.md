# Old-Photo-Restoration-with-InstructPix2Pix-via-Lightweight-LoRA-Fine-Tuning
This project addresses the challenge of restoring old, degraded photographs (specifically removing scratches and noise) while preserving their original structure and historical texture. We utilize **InstructPix2Pix (IP2P)**, a text-guided image editing model, and fine-tune it using **Low-Rank Adaptation (LoRA)**.
